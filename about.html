What is twa.sh?

  • @berkes' mirror of Dabr. Adapted code on github
  • A mobile web interface to Twitter's API
  • An open source project by @davidcarrington with inspirations from @whatleydude
  • Secure, storing your Twitter login details in an encrypted cookie on your machine, and never stored on the website
  • Twash is monitored with my private installation of piwik, for statistics. Logs, stats or other information will never be handed over or sold to anyone, unless forced by the Dutch law

If you have any comments, suggestions or questions then feel free to get in touch with @berkes.
